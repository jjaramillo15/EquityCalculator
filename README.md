# Texas Hold'em Equity Calculator

A monolithic Next.js baseline for a Texas Hold'em equity calculator MVP.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Vitest and Testing Library
- Playwright

## Available scripts

- `npm run dev`
- `npm run build`
- `npm test`
- `npm run test:e2e`

## Bootstrap scope

This bootstrap adds:

- the initial App Router shell
- a smoke test for the home page scaffold
- Tailwind, Vitest, and Playwright configuration
- baseline environment and ignore files

## Next step

Install the dependencies with your preferred package-management flow in a full Node environment, then run:

```bash
npm test -- src/app/page.test.tsx
npm run build
```
